== Description ==

**One prompt = Complete Elementor page!** Powered by xAI's Grok AI.

**Features:**
* AI-powered page generation
* Elementor widget integration
* Automatic updates from GitHub
* Settings page with API key management
* PHP 8.2+ compatible

Get your FREE xAI API key at https://x.ai/api

== Installation ==

1. Upload the plugin ZIP
2. Activate the plugin
3. Get your xAI API key at https://x.ai/api
4. Go to **Settings > JagGrok Elementor** and enter your API key
5. Click **Test Connection**
6. Start building AI-powered pages in Elementor!
