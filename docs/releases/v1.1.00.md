# AiMentor Elementor v1.1.00 Draft Release Notes

## Summary
- Automates daily provider health checks with WP-Cron so stored API credentials are re-validated without manual interaction.
- Tracks consecutive connection failures per provider, resetting on success and surfacing alerts when the error threshold is reached.
- Adds settings toggles and recipient controls for administrators to manage scheduled checks and notification routing directly in WordPress.

## Testing
- php -l aimentor-elementor.php
- php -l includes/settings.php
- php -l includes/settings-template.php
